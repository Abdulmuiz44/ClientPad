# ClientPad Cloud

ClientPad Cloud is the hosted path for teams that want managed ClientPad infrastructure without running every server themselves.

Cloud adds operator auth, workspace bootstrap, API key management, usage tracking, readiness diagnostics, and Lemon Squeezy checkout on top of the same open-source packages.

## What Cloud includes

- Operator accounts and sessions.
- Workspace and starter project bootstrap.
- API key management.
- Cloud readiness telemetry.
- WhatsApp diagnostics.
- Usage tracking for hosted billing.
- Lemon Squeezy checkout and customer portal links.

## Domains

- Dashboard: `https://app.clientpad.xyz`
- Cloud API: `https://api.clientpad.xyz/api/cloud/v1`
- Public API: `https://api.clientpad.xyz/api/public/v1`
