Self-hosting

Self-hosted ClientPad deployments run your PostgreSQL database, migrations, server package, and dashboard. API keys can remain free and unlimited because you own the infrastructure.

1. Database

Create a PostgreSQL database and set `DATABASE_URL`.

2. Migrations

Run `clientpad migrate` from the CLI.

3. API keys

Create a workspace API key with `clientpad api-key create`.

4. Server

Mount `createClientPadHandler` in a fetch-compatible runtime.

5. Dashboard

Point the dashboard at your Cloud/API URL.

6. WhatsApp

Add the WhatsApp webhook handler when you are ready for live conversations.
