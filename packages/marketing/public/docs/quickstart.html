Quickstart

Use the SDK against hosted Cloud or a self-hosted public API.

pnpm add @clientpad/sdk
import { ClientPad } from "@clientpad/sdk";

const clientpad = new ClientPad({
  baseUrl: "https://api.clientpad.xyz/api/public/v1",
  apiKey: process.env.CLIENTPAD_API_KEY!,
});

await clientpad.leads.create({
  name: "Ada Customer",
  phone: "+234...",
  source: "Website",
});

Hosted path

Create a workspace in the dashboard, copy a server-side API key, and point the SDK at `api.clientpad.xyz`.

Self-hosted path

Run PostgreSQL, apply migrations, create an API key with the CLI, and point the SDK at your own public API route.
